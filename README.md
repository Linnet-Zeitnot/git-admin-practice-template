# Практика Git

## Что нужно сделать
1. Заполнить proof/windows.txt выводом команд из Windows.
2. Заполнить proof/ubuntu.txt выводом команд из Ubuntu.
3. Создать ветки feature-windows и feature-ubuntu.
4. Сделать merge обеих веток в основную.
5. Проверить, что во вкладке Actions стоит зелёная галка.
